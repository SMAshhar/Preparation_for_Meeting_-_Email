# prep-for-a-meeting

A lightweight Python demo that uses CrewAI agents and tasks to prepare for a meeting. It asks for participants, meeting context, and objectives, then runs research, industry analysis, strategy, briefing, and email-writing steps.

## Setup

1. Install Python 3.11 or newer.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Create a `.env` file if needed for your API keys.
4. Run:

   ```bash
   python main.py
   ```
